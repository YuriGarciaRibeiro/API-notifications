# Drift Severity

## Bloqueante

- Comando documentado falha e impede setup básico.
- Caminho essencial do projeto não existe.

## Relevante

- Estrutura documentada desatualizada causa retrabalho.
- Guia menciona recurso removido/renomeado.

## Cosmética

- Texto ou exemplo secundário desatualizado sem impacto de execução.
